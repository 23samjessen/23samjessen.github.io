# Private recommendation letter workflow

Keep the **full unredacted Hitachi recommendation letter** out of the public website and out of the public GitHub repository.

## What stays public
- A short summary card on the website
- A **Request access by email** button
- Public certificates only

## What stays private
- The full recommendation PDF
- The Google Drive sharing link
- Any internal notes about who has been granted access

## Suggested local files
- `Recommendationletter_SamJessen.pdf`
- `google-drive-link.private.txt`
- `granted-access.private.csv`

## Manual approval flow
1. Keep the Google Drive file set to **Restricted**.
2. When someone requests the letter from the site, review the request.
3. Share the Drive file manually with that person's work email.
4. If needed later, remove their access in Google Drive.

## Important
The public website is intentionally designed to show a **summary + request access by email** instead of exposing the full letter or direct manager contact details.
